java实现在线预览PDF文件

最近碰到一个需求，要求实现office文件和PDF文件能够实现实时预览的效果

最后经过查找发现现在有新的头部信息application/pdf

现在主流浏览器均支持这种资源的MIME类型 ( 测试了ie会直接进行下载 )
